
# Desafio do projeto bytebank

Projeto realizado no desafio do curso alura utilizando as tecnologias do spring batch da classe pagamento dos funcionarios, lendo e importando um arquivo dadosficticios.csv da pasta files executando um processamento de lote de dados com Spring Batch para integrar leitura e escrita no Job usando o banco de dados Postgre SQL.


## Tecnologias usadas

- Java 21
- Spring Boot versão 3.5.11
- Spring Batch
- Spring Data JPA
- Banco de dados Postgre SQL
